
# AI Interview Assistant Setup

## Requirements
- Python 3.8+
- Install dependencies:
  pip install openai whisper torch sounddevice numpy wavio tkinter

## OpenAI API
- Get your API key from https://platform.openai.com/account/api-keys
- Replace "YOUR_OPENAI_API_KEY" in the scripts with your actual key

## Scripts
- whisper_gpt_assistant.py: Continuous listening in terminal
- gui_assistant.py: Simple GUI version

## Notes
- Adjust recording duration if needed
- Upgrade Whisper model for better transcription ("small", "medium", "large")
